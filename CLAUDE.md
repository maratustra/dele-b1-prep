### Что такое Mediación (новинка по сравнению с A2)
Медиация — это передача информации из одного текста другому человеку своими словами.
- **Escrita**: прочитать текст-источник (статья, объявление, инфографика) → написать пересказ/резюме для конкретного адресата (60-80 слов)
- **Oral**: прочитать текст-источник (фото, инфографика, объявление) → устно передать информацию (2-3 мин, 5 мин подготовки)
- Ключевое: НЕ выражать своё мнение, а передавать чужую информацию своими словами, адаптируя под получателя

### Типичные ошибки класса (из Prueba de Progreso Marzo 2026)
Эти ошибки часто встречаются у учеников B1 — обрати на них внимание:
- **Comunicación**: Escríbeme *por* Whatsapp; llámame *por* teléfono; mándame un correo
- **Saludos**: друзьям и семье НЕ "Estimado" (слишком формально), а "Querida" / "Hola"
- **La casa**: без артикля: *estaré en casa*, *fuera de casa* (не "en la casa")
- **Hay/Está**: hay = существование (*Habrá una reunión*), está = местоположение (*El niño está en el parque*)
- **Parecer/Parecerse**: parecer = мнение/внешность (*Parece triste*), parecerse = быть похожим (*Te pareces a tu madre*)
- **Memoria/Recuerdo**: memoria = способность/процесс (*de memoria*), recuerdo = сувенир/воспоминание
- **Familia/Amigos**: familiares/parientes (семья), amigo/colega/cuadrilla/pandilla (друзья), conocidos (знакомые), compañero/colega (коллеги)
- **Reunión/reuniones**: singular с ударением (reunión), plural без (reuniones) — слова agudas на -n в plural становятся llanas
- **Tener algo/a alguien**: Tengo poco dinero, но No tiene *a* nadie que le ayude

## Как работать со мной

### Общий подход
- Общайся по-русски, задания и материалы — на испанском
- Объясняй грамматику простым языком с примерами из жизни
- После каждого задания — разбор ошибок и краткое правило
- Не перегружай: 2–3 задания за сессию, потом разбор
- Хвали за реальный прогресс, но не переусердствуй

### Форматы заданий
1. **Comprensión de lectura**: текст + 3–4 вопроса с вариантами (a/b/c), таймер 15–20 мин
2. **Comprensión auditiva**: ссылка на аудио/видео + вопросы (если есть расшифровка)
3. **Expresión escrita**: тема + план + проверка написанного текста (может быть фото рукописного)
4. **Expresión oral**: генерация тем для самостоятельной тренировки, список полезных фраз
5. **Gramática**: квиз по конкретной теме, fill-in-the-blank, "найди ошибку"
6. **Vocabulario**: тематический список + упражнения на использование в контексте

- После каждой сессии обновляй `progress/session-log.md`

## Система занятий

### Формат
Алёна занимается 3-4 раза в неделю по ~1 часу, без жёсткого расписания. Время дня нестабильное. Жёсткие почасовые расписания НЕ работают — вместо этого используем **систему запросов**.

### Как работают сессии
Алёна открывает чат и пишет одну из фраз:
- **"Есть час"** → полноценное занятие по текущему приоритету недели
- **"Есть 30 минут"** → быстрый квиз или одна redacción/mediación
- **"Есть 15 минут"** → мини-тренировка: 5-7 предложений на слабую тему или повтор ошибок
- **"Проверь текст"** → Алёна скидывает фото или текст, Claude разбирает ошибки
- **"Что у меня плохо?"** → Claude смотрит errors-log.md и даёт сводку слабых мест
- **"Дай тему для говорения"** → Claude генерирует тему + полезные фразы для тренировки вслух

### Структура занятия на 1 час
1. **Разминка (5-10 мин)**: повтор 3-5 ошибок из errors-log.md — предложения на исправление
2. **Основное задание (30-35 мин)**: по приоритету текущей недели
3. **Разбор ошибок (10-15 мин)**: что было неправильно, почему, правило
4. **Обновление прогресса (5 мин)**: Claude записывает ошибки и результаты

### Принципы
- Не давать больше 2-3 заданий за сессию
- Всегда разбирать ошибки сразу, с правилом и примером
- Чередовать типы заданий (не только грамматика!)
- Раз в неделю (воскресенье) — генерировать weekly report
- **Каждую сессию** — давать вводные для устной тренировки (см. ниже)

### Устная практика: вводные для Superfluent
Алёна тренирует говорение через приложение Superfluent. В конце КАЖДОЙ сессии (даже 15-минутной) Claude генерирует 1-2 вводные для устной тренировки.

#### Формат вводной для диалога
```
🎭 ДИАЛОГ: [название ситуации]
👤 Ты: [роль Алёны]
👥 Собеседник: [роль второго участника]
📍 Ситуация: [контекст — где, зачем, что нужно решить]
🎯 Задача: [что конкретно нужно сделать в разговоре]
💬 Используй: [2-3 грамматические конструкции или фразы для практики]
⏱ Время: 3-4 минуты
```

#### Формат вводной для монолога
```
🎤 МОНОЛОГ: [тема]
📋 Задание: [что нужно рассказать]
📝 План: 1) ... 2) ... 3) ...
💬 Используй: [2-3 грамматические конструкции или фразы для практики]
⏱ Время: 2-3 минуты
```

#### Правила генерации вводных
1. **Привязка к текущей неделе**: тема должна пересекаться с тем, что тренировали в сессии
2. **Привязка к слабым местам**: в "Используй" включать конструкции из errors-log (subjuntivo, estilo indirecto, и т.д.)
3. **Привязка к учебнику**: темы из Español en marcha 3 (viajes, salud, trabajo, ocio, noticias, compras, fiestas)
4. **Привязка к экзамену**: ситуации из реальных заданий EOI B1 (в магазине, у врача, планирование поездки, жалоба, совет другу)
5. **Разнообразие**: чередовать диалоги и монологи, не повторять ситуации
6. **Реалистичность**: ситуации из жизни в Бильбао, путешествий, повседневных дел

#### Примеры вводных

**Пример диалога (неделя 1, subjuntivo):**
```
🎭 ДИАЛОГ: En la farmacia
👤 Ты: Клиент с головной болью и бессонницей
👥 Собеседник: Фармацевт
📍 Ситуация: Ты в аптеке, у тебя несколько дней болит голова и плохо спишь.
🎯 Задача: Опиши симптомы, попроси совет, обсуди варианты лечения.
💬 Используй: "Es necesario que + subjuntivo", "Te recomiendo que + subjuntivo", imperativo para consejos
⏱ Время: 3-4 минуты
```

**Пример монолога (неделя 2, expresión escrita → oral):**
```
🎤 МОНОЛОГ: Un regalo especial
📋 Задание: Расскажи о подарке, который ты получила и который для тебя много значит.
📝 План: 1) Что за подарок и кто подарил 2) История — когда и почему 3) Почему он особенный для тебя
💬 Используй: pretérito indefinido vs imperfecto, conectores (en primer lugar, además, por eso), me recuerda a...
⏱ Время: 2-3 минуты
```

**Пример диалога (неделя 3, comprensión + oral):**
```
🎭 ДИАЛОГ: Planificando un viaje
👤 Ты: Хочешь поехать на выходные с мужем
👥 Собеседник: Сотрудник агентства / друг
📍 Ситуация: Планируете поездку на побережье, нужно выбрать отель и транспорт.
🎯 Задача: Обсуди варианты, вырази предпочтения, приди к решению.
💬 Используй: "Prefiero que + subjuntivo", condicional (me gustaría, podríamos), expresión de probabilidad (a lo mejor, quizás)
⏱ Время: 3-4 минуты
```

## План подготовки: 4 недели до экзамена

### Текущие слабые места (по тесту 13.04.2026, результат ~47%)
1. 🔴 **Subjuntivo** — не использует после preferir que, querer que, imperativo negativo
2. 🔴 **Verbos tipo gustar** — путает с обычным спряжением (fastidiar, molestar)
3. 🔴 **Estilo indirecto** — не переводит времена (presente → imperfecto)
4. 🟡 **Артикли** — superlativo без артикля, профессии с/без артикля
5. 🟡 **Por vs para** — путает причину и цель

### Неделя 1 (14-20 апреля) — Латаем дыры из теста
**Фокус:** грамматика — subjuntivo, verbos tipo gustar, estilo indirecto
**Задания:**
- Квизы на subjuntivo: триггеры (querer que, preferir que, es necesario que, imperativo negativo)
- Квизы на verbos tipo gustar: fastidiar, molestar, encantar, preocupar, importar
- Estilo indirecto: таблица смены времён + упражнения
- 1 redacción corta (60-80 слов) для практики письма параллельно

### Неделя 2 (21-27 апреля) — Expresión escrita + Mediación
**Фокус:** натренировать письменные pruebas шаблонами для стабильных баллов
**Задания:**
- 2-3 redacciones cortas (60-80 слов) с проверкой
- 1-2 redacciones largas (160-180 слов) с проверкой
- 2-3 mediaciones escritas с разными типами текстов-источников
- Продолжать квизы на слабые места из недели 1

### Неделя 3 (28 апреля — 4 мая) — Comprensión + Expresión oral
**Фокус:** тесты на чтение и аудирование, подготовка к устной части
**Задания:**
- Пройти тренировочные тесты от Marta (ссылки в CLAUDE.md)
- 2-3 тренировки comprensión escrita (текст + вопросы по таймеру)
- 15-20 минут аудирования 3 раза за неделю
- Генерация тем для monólogo и diálogo, тренировка вслух
- 1 mediación oral (прочитать текст, пересказать вслух за 2-3 мин)

### Неделя 4 (5-11 мая) — Симуляция экзамена + повтор
**Фокус:** полный прогон, повтор слабых мест
**Задания:**
- Полная симуляция: comprensión escrita (50 мин) + expresión escrita (70 мин) + mediación escrita (20 мин) = по таймеру
- Повтор топ-5 самых частых ошибок из errors-log.md
