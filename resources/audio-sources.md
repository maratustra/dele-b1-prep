# Аудио и видео ресурсы для тренировки аудирования

> Подбирай материалы по своему уровню. Начинай с A2-B1, постепенно переходи к B1-B2.
> Главное — регулярность: 15-20 минут каждый день.

## Ресурсы с субтитрами и расшифровками (идеально для тренировки)

### YouTube каналы
- **SGEL ELE** — видео к учебнику Español en marcha (ищи "Español en marcha 3")
- **ProfeDeELE** — подкасты и видео B1 с транскрипцией: https://www.profedeele.es
- **Español con Juan** — простые объяснения грамматики и разговорный испанский
- **Dreaming Spanish** — метод comprehensible input, есть уровень Intermediate
- **SpanishPod101** — структурированные уроки с субтитрами
- **Easy Spanish** — уличные интервью с субтитрами (español + английский)

### Подкасты
- **News in Slow Spanish** — новости на медленном испанском с расшифровкой
- **Hoy Hablamos** — ежедневный подкаст для изучающих, транскрипции на сайте
- **Radio Ambulante** — NPR на испанском, есть расшифровки (уровень B1-B2)
- **Notes in Spanish** — Intermediate уровень, есть транскрипции (платно)

### Сайты с упражнениями на аудирование
- **Instituto Cervantes: AVE** — https://ave.cervantes.es (упражнения по уровням)
- **Ver-taal.com** — аудирование с упражнениями: https://www.ver-taal.com
- **Videoele.com** — видео с упражнениями для B1: http://www.videoele.com
- **Practica Español** — новости для изучающих: https://www.practicaespanol.com

## Как тренироваться с аудио

### Формат тренировки (20 мин)
1. **Первое прослушивание** (без паузы) — пойми общий смысл
2. **Вопросы** — прочитай и ответь по памяти
3. **Второе прослушивание** — проверь и уточни ответы
4. **Расшифровка** — прочитай текст, найди слова, которые не понял
5. **Запиши** — новые слова в vocabulary.md

### Еженедельная цель
- Минимум 3 аудио-сессии в неделю по 15-20 мин
- 1 полный тест comprensión auditiva (формат DELE) в неделю
