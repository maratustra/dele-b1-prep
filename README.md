# DELE B1 Exam Preparation 🇪🇸

Персональная система подготовки к экзамену DELE B1 с использованием Claude Code.

## Структура

```
dele-b1-prep/
├── CLAUDE.md              — инструкции для Claude Code
├── materials/             — учебные материалы
│   ├── textbook/          — учебник Español en marcha 3
│   ├── exams/             — примеры экзаменов DELE B1
│   └── class-notes/       — материалы с уроков
├── progress/              — трекинг прогресса
│   ├── topics-tracker.md  — статус по темам
│   ├── errors-log.md      — журнал ошибок
│   ├── session-log.md     — лог сессий
│   ├── vocabulary.md      — словарь
│   └── weekly-reports/    — еженедельные отчёты
├── exercises/             — сгенерированные задания
│   ├── lectura/
│   ├── escritura/
│   ├── auditiva/
│   ├── oral/
│   └── grammar/
└── resources/             — справочные материалы
    ├── exam-tips.md       — лайфхаки для экзамена
    ├── useful-phrases.md  — фразы-клише
    └── audio-sources.md   — ссылки на аудио/видео
```

## Как пользоваться

1. Открой терминал в папке проекта
2. Запусти `claude` (Claude Code)
3. Claude прочитает `CLAUDE.md` и будет знать весь контекст
4. Попроси сгенерировать задание, проверить текст или обновить прогресс

## Примеры команд для Claude Code

- "Сгенерируй тест по грамматике Unidad 3 (oraciones de relativo)"
- "Проверь мой текст на ошибки" (вставить текст)
- "Покажи мой прогресс за эту неделю"
- "Какие у меня самые частые ошибки?"
- "Дай задание на comprensión de lectura по теме viajes"
